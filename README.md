# MusicKiller

一个简洁好用的音乐播放App，包含前后端代码，编译后可直接使用。包含在线歌单、排行榜、歌词、搜索等！支持无损音质

## 注意

只能用于学习用途

## 技术栈

- Kotlin
- Java

## 构建

需要自行修改签名 `/app/build.gradle` 修改签名路径和账号密码
